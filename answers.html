What did we learn today?


Elements of design are extremely important when designing a webpage!



When a web designer is creating a webpage it is really important for them to think of the page from a user's perspective. Everything should be kept relatively simple so that each page is easy to use. Information should be displayed clearly and concisely so that it is easy for the user to read. Images should be displayed in a clean way and should not take away from information on the page. Text should not be written in large blocks, but instead spread throughout the page.
Hopefully this information helped you uunderstand more about elements of design and how they are implemented in pages you look at everyday without even noticing sometimes!
